# Joyonto Rozario Portfolio

A refined, production-ready static portfolio website optimized for SEO, accessibility, responsiveness, and performance while preserving the existing visual identity and layout.

## Files
- index.html — main content and metadata
- styles.css — optimized styling
- script.js — accessible navigation behavior
- robots.txt — search engine crawling instructions
- sitemap.xml — sitemap for indexing
- manifest.json — web app metadata

## Preview locally
Run:

```bash
python -m http.server 8000
```

Then open http://localhost:8000/
